# Usage

## Install
```sh
npm install
```

## Develop
```sh
npm run dev
```

## Build
```sh
npm run build
```

## Lint
```sh
npm run lint
```

## Examples
- `npm run dev` for local development.
- `npm run build` for production bundles.

## Edge Cases
- Ensure `.env` exists if required by the app.
- Some commands may need backend services running.
